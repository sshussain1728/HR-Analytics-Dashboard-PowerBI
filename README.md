# HR-Analytics-Dashboard-PowerBI
Interactive HR Analytics dashboard built using Power BI with data cleaning and preprocessing performed in MySQL Workbench to analyze employee attrition, workforce demographics, and performance metrics.
# HR Analytics Dashboard using Power BI

## Project Overview

This project presents an HR Analytics Dashboard developed in Power BI to help organizations analyze employee data, identify workforce trends, and understand factors contributing to employee attrition.

## Key Features

- Employee Attrition Analysis
- Department-wise Employee Distribution
- Gender Diversity Analysis
- Job Role Analysis
- Age Group Distribution
- Salary Analysis
- Employee Performance Insights

---

## Data Cleaning

The dataset was cleaned and prepared using MySQL Workbench before importing it into Power BI. Cleaning steps included:

- Removing duplicate records
- Handling missing values
- Standardizing text fields
- Validating salary values
- Preparing data for dashboard visualization
